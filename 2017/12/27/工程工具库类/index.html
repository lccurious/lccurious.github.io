工程工具库类

发表于 2017-12-27 更新于 2020-10-01 分类于 工程 阅读次数： 1k 1 分钟

工程工具库，其实很多就是由一些组织或者个人写成的提供给其他开发人员使用的函数库，我们常常称这些函数库为“轮子”，而我们在很多项目的构建中就会成为使用轮子的人。在这篇文章中将会简单介绍一些常用的C++工具库。

Boost库

Boost库是一个可移植、提供源代码的C++库，作为标准库的后备，是C++标准化进程的开发引擎之一。 Boost库由C++标准委员会库工作组成员发起，其中有些内容有望成为下一代C++标准库内容。在C++社区中影响甚大，是不折不扣的“准”标准库。Boost由于其对跨平台的强调，对标准C++的强调，与编写平台无关。大部分boost库功能的使用只需包括相应头文件即可，少数（如正则表达式库，文件系统库等）需要链接库。但Boost中也有很多是实验性质的东西，在实际的开发中使用需要谨慎。

LibTiff库

这是一个由C语言开发的，支持读写Tag Image File Format（TIFF）的软件。给其他开发人员在各个32位和64位平台上提供读写TIFF文件数据的函数工具。

OpenCV库

OpenCV的全称是Open Source Computer Vision Library，是一个跨平台的计算机视觉库。OpenCV是由英特尔公司发起并参与开发，以BSD许可证授权发行，可以在商业和研究领域中免费使用。OpenCV可用于开发实时的图像处理、计算机视觉以及模式识别程序。该程序库也可以使用英特尔公司的IPP进行加速处理。为了推进机器是觉得研究，提供一套开源且优化的基础库，不需要重新造轮子。

CUDA库

基于Nvidia显卡的库，由Nvida公司开发，提供给开发人员用于高效使用Nvida显卡的高效底层库，针对显卡体系结构进行很特别的优化。

VTK库

可视化工具库（VTK， Visualization Toolkit）是一个开放源代码，跨平台、支持平行处理（VTK曾用于处理大小近乎1个Petabyte的数据，其平台为美国Los Alamos国家实验室所有的具1024个处理器之大型系统）的图形应用库。2005年时曾被美国陆军研究实验室用于即时模拟俄罗斯制反导弹战车ZSU23-4受到平面波攻击的情形，其计算节点高达2.5兆个之多。

Qt开发

从个人角度出发Qt的整个编程环境还是非常友好的，所有的工程编译由qmake负责，通过.pro文件来指定编译规则，通过信号槽机制完成各类事件触发的设计，界面的设计也是可视化的。对应的工程都是可以跨平台的，大大提高了编写设计效率。
